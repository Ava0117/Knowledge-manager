# AI学习平台 - 规格说明书

## 1. 项目概述

**项目名称**: AI智能学习助手
**项目类型**: 交互式Web学习平台
**核心功能**: 基于AI的个性化学习平台，支持PPT/PDF学习、AI教师语音解说、交互式学习组件、实时测评与反馈
**目标用户**: 学生、备考人员、知识学习者

## 2. 视觉与界面规格

### 整体视觉风格
- **设计语言**: 现代教育科技风格，融合毛玻璃效果与渐变色彩
- **色彩方案**:
  - 主色: #6366F1 (靛蓝色 - 代表智慧与专注)
  - 次色: #8B5CF6 (紫罗兰 - 代表创造力)
  - 强调色: #10B981 (翡翠绿 - 正确/成功)
  - 警示色: #F59E0B (琥珀色 - 警告)
  - 错误色: #EF4444 (红色 - 错误)
  - 背景: #0F172A (深蓝黑色) 到 #1E293B (深灰蓝) 渐变
  - 文字: #F8FAFC (亮白) 到 #94A3B8 (灰蓝) 渐变

### 布局结构
- **顶部导航栏**: Logo、功能切换标签页(学习/测验/题库/复习计划)
- **主工作区**:
  - 左侧: 内容展示区(PPT/PDF预览、聚光灯效果)
  - 右侧: AI交互面板(语音控制、实时反馈)
- **底部工具栏**: 播放控制、进度指示、音量调节

### 视觉效果
- **聚光灯效果**: 内容区域半透明遮罩，仅高亮当前讲解区域
- **毛玻璃卡片**: 所有面板使用 backdrop-filter: blur(12px)
- **动画效果**: 渐入动画300ms ease-out、悬停缩放1.02倍
- **阴影**: 0 4px 20px rgba(99, 102, 241, 0.3)

## 3. 功能模块规格

### 3.1 内容上传与解析
- **支持格式**: PPT(.pptx), PDF(.pdf)
- **上传方式**: 拖拽上传 + 点击选择文件
- **解析处理**:
  - PDF: 使用pdf.js提取文本和页面
  - PPT: 提取幻灯片内容转为图片序列
- **预览展示**: 分页预览，支持缩放

### 3.2 AI教师语音解说
- **语音引擎**: Web Speech API (SpeechSynthesis)
- **音色选择**: 3种音色(标准女声、温和男声、活力女声)
- **语速调节**: 0.5x - 2.0x
- **功能控制**: 播放/暂停/停止/重播
- **高亮同步**: 语音播放时对应内容高亮显示

### 3.3 聚光灯效果
- **效果类型**: 径向渐变遮罩 + 内容区域半透明
- **可调节**: 聚光灯大小(50px-300px)
- **跟随模式**: 可选跟随鼠标或固定位置

### 3.4 交互式学习组件

#### 流程图模块
- **节点类型**: 开始/结束(圆角)、处理(矩形)、判断(菱形)、输入输出(平行四边形)
- **交互操作**: 点击节点查看详细解释
- **连接线**: 带箭头的贝塞尔曲线

#### 模拟器模块
- **内置模拟器**:
  - 数学计算模拟器(函数图像、可视化计算过程)
  - 公式推导模拟器(分步展示推导过程)
- **交互操作**: 滑块调节参数、实时更新结果

### 3.5 互动测验系统
- **题目类型**: 单选、多选、判断、填空、简答
- **AI评分**: 基于语义分析的智能评分
- **反馈机制**:
  - 即时反馈(正确/错误)
  - 详细解析
  - 相关知识点链接
- **进度追踪**: 正确率、用时、弱项分析

### 3.6 题目上传与讲解
- **上传方式**: 文本输入/图片上传(OCR识别)
- **AI讲解**: 分步解析、知识点拆解
- **反馈生成**: 错误原因分析、改进建议

### 3.7 复习计划与备考
- **计划生成**: 基于错题库的个性化复习计划
- **记忆曲线**: 根据遗忘曲线安排复习时间
- **大考倒计时**: 期末考日期设置、倒计时显示
- **薄弱点追踪**: 自动记录高频错误知识点

## 4. 技术实现

### 前端技术栈
- **框架**: 原生HTML5 + CSS3 + JavaScript ES6+
- **语音API**: Web Speech API
- **文件解析**: PDF.js (Mozilla开源库)
- **图表渲染**: Canvas API
- **存储**: LocalStorage (本地数据持久化)

### 外部资源
- **图标库**: Font Awesome 6.4.0
- **字体**: Noto Sans SC (思源黑体中文优化版)
- **动画库**: 纯CSS动画
